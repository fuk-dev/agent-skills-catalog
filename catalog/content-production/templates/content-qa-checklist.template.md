# コンテンツ品質QAチェックリスト

公開前に必ずこのQAを通すこと。AI生成文にありがちな「一般論化・誇張・ハルシネーション・法令リスク・独自性不足」を出稿前につぶすためのチェックリスト。すべての項目に `[x]` が付くまで公開しない。

## 必須チェック

- [ ] AIだけの一般論になっていないか
- [ ] 一次情報（自社の実績・データ・現場の声）が入っているか
- [ ] 未確認の数値がないか（出典のない統計・割合・金額を書いていないか）
- [ ] 実在しない事例を作っていないか（架空の顧客・症例・受賞歴など）
- [ ] 効果を保証していないか（「必ず」「100%」「絶対治る」等の断定表現がないか）
- [ ] 医療広告・薬機法・景表法などに抵触していないか
- [ ] FAQは実際の問い合わせに基づいているか
- [ ] 外部口コミを転載・選別・編集していないか（医療: 治療内容・効果の体験談は転載不可）
- [ ] 事業者関与のある推奨・レビュー表示に関与の明示があるか（ステマ告示・2023年10月施行）
- [ ] CTAが本文から自然につながっているか
- [ ] 文章が読みやすいか（一文が長すぎない・見出し構造が適切）
- [ ] タイトルと本文の内容が一致しているか
- [ ] 検索意図に答えているか（読者の知りたいことに着地しているか）

## 業種別の追加注意

### 医療
- [ ] 治療内容・効果に関する体験談を掲載していないか（掲載不可・条件なし。外部口コミの転載も含む）
- [ ] ビフォーアフター写真に5条件（治療内容・治療期間回数・費用・リスク副作用・問い合わせ先）の併記があるか
- [ ] 最上級・比較優良表現（「最先端」「最高」等）がないか
- [ ] 効果効能の断定、未承認薬・自由診療の誇大表現を避ける

### 士業（弁護士・税理士・社労士など）
- [ ] 「必ず勝てる」「100%還付」等の結果保証をしない
- [ ] 法改正の最新性を確認（古い制度・税率を書いていないか）

### 金融
- [ ] 元本保証・利回り断定など金商法・景表法に触れる表現がないか
- [ ] リスク説明を省略していないか

### 建設・不動産
- [ ] 「完全」「絶対安全」等の不当表示、誇大広告がないか
- [ ] 価格・面積・立地などの数値は実データに基づくか

### 飲食・美容
- [ ] 「痩せる」「アンチエイジング」等の効果断定（景表法・薬機法）がないか
- [ ] メニュー・価格・栄養表示が実際と一致しているか
