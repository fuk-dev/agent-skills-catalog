# SEO記事テンプレート（4型）

> プロセス4で使用。SEO 記事の量産に使う4つの型（ハウツー・比較・解説・事例）のテンプレ。

---

## 使い方

1. 記事の検索意図を判定
2. 該当する型を選択
3. テンプレートに沿って記事を執筆
4. 執筆前に必ず `references/seo-content-strategy.md` のキーワード戦略を確認

---

## 記事執筆時の共通チェックリスト

すべての SEO 記事で共通：

```
【SEO】
[ ] title タグは 32 字以内で主要キーワード含む
[ ] meta description は 90-120 字
[ ] h1 は 1 個のみ、主要キーワード含む
[ ] h2 に関連キーワード含む（3-5 箇所）
[ ] 主要キーワードを自然に 5-10 回
[ ] 内部リンク 3-5 箇所
[ ] 外部リンクは信頼できるソースのみ

【構造】
[ ] 冒頭 200 字以内で結論
[ ] 見出しで全体像が伝わる
[ ] 画像に alt 属性
[ ] 箇条書き・表で視認性向上

【品質】
[ ] 誤字脱字なし
[ ] 文体統一
[ ] 根拠ある数値
[ ] 業種の法令遵守
[ ] NG ワード不使用

【CV】
[ ] CTA が末尾に配置
[ ] 関連サービスへの内部リンク
[ ] 関連記事へのリンク
```
